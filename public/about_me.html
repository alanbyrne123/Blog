Hello, I'm Alan

I built this small site with accessibility and simplicity in mind.

This blog is a personal project to share my thoughts on modern tech and practice my web dev skills.



What I learned so far

  • How to store and retrieve posts with localStorage and keep the newest posts on top.
  • how to create a responsive navigation, some hover interactions, and a persistent dark mode.
  • How to create a simple email form that uses the user's email client.
  • How to use JavaScript to create a modal image viewer and blog post forum.


Features

  • Create and delete persistent blogs.
  • Interactive hover UI navbar and a user friendly webpage.
  • A persistent dark mode with font size changing.
